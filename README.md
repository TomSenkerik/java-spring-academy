# Readme

Learning app of Java and Spring Boot. App contains:

- Actuator autoconfig
- Lombok
- Marshaling and Unmarshaling data with FasterXML Jackson
- Redis, RabbitMQ and MongoDB autoconfig
- Default validation
- Endpoint documentation with OpenAPI
